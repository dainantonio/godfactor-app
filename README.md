# GodFactor - Christian Community Platform

A peaceful social platform for Christians to connect through daily devotionals, prayer, and encouragement without debate or division.

## Quick Start

1. **Clone or download** this repository
2. **Install dependencies**: `npm install`
3. **Run the app**: `npm start`
4. **Open browser**: Go to `http://localhost:3000`

## Default Login
- **Admin**: `admin@godfactor.app` / `admin123`
- **Regular users**: Sign up for free account
